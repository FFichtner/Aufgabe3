===== Allgemeines =====

__Was ist GitHub__?

GitHub ist ein Online-Dienst welcher euch Softwareentwicklungsprojekte auf einem Server ermöglicht. (**Filehosting**). Namensgeben war hierbei das Versionsverwaltungstool **git**. 
Damit könnt ihr dann in eurem eigenen **Repository** als Team untereinander kommunizieren, die Fortschritte des Projektes einsehen und euch bei Problemen unterstützen, am Ende könnt ihr dann euer Projekt mergen und abgeben.

Übrigens bedeutet das Wort "git" im englischen so viel wie "Blödmann".

===== Einführung und Hello World =====


__Wie benutze ich GitHub?__

Auf GitHub werden auch Tutorials bzw. Guides angeboten, die euch in weniger Minuten mit der Software vertraut machen.
Den Einsteigerkurs findet ihr [[https://guides.github.com/activities/hello-world/|hier]].
Keine Sorge die Tutorials sind meistens für absolute Neueinsteiger verfasst, ihr dürftet also keine Probleme haben, dass allzeit beliebte "Hello World" hochzuladen.
