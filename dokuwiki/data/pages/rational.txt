====== Rational Software Architect ======

===== Allgemeines =====

Rational Software Architect ist eine IDE für **UML**(Unified Modeling Language) zum Architekturdesign von C++ oder Java EE Anwendungen sowie Web Services. 
Die Software wird von **IBM** zur Verfügung gestellt.

[[http://www-03.ibm.com/software/products/de/ratsadesigner|hier]] könnt ihr euch die Software für 30 Tage kostenlos herunterladen und testen.
